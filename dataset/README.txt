드론 수색 경로 수립 데이터

입력
heatmap_values
SRU.csv

출력
경로의 이동을 나타내는 vertex(시작점과 도착점을 포함하는 경로가 꺾이는 점. example.png 참조)

vertex는 두 가의 형태로 출력할 수 있습니다. (택 1)
1. 히트맵 각 셀의 방문 순서를 나타내는 cell index array (heatmap_index.csv 파일 기반)
2. vertex의 좌표(위도, 경도) array (heatmap_center_point.csv 파일 기반)


파일 설명
기본적으로 히트맵은 20 X 20 크기이며, 각 셀은 해당 셀에 포함된 입자(수색 대상)의 개수를 값으로 가집니다.
히트맵 한 셀의 크기는 가로 1.03km / 세로 0.5km 입니다.
제공드린 모든 csv 파일은 ','를 구분점으로 가집니다.

1. heatmap_index.csv 
   히트맵 각 셀을 구분하기 위한 index를 나타내는 파일입니다.
   index는 x/y 로 나타나며, 왼쪽으로 갈수록 x값이 작고, 아래로 내려갈수록 y값이 작습니다.
   x와 y는 모두 0-19의 범위를 가집니다.
   드론의 수색 경로가 히트맵 각 셀의 방문 순서로 표현된다면, 해당 파일을 참고하시면 좋을 것 같습니다.


2. heatmap_center_point.csv 
   히트맵의 각 셀의 중심점 좌표를 나타내는 파일입니다.
   csv 구분기호는 ','이며, '/'를 통해 경도와 위도를 구분합니다. 
   예를 들어, index 19, 0의 셀이 "133.45206871/38.294637095" 인 경우, 
                  19, 0 셀의 경도는 13133.45206871, 위도는 38.294637095입니다.
   수색 경로 vertex의 좌표를 출력하실 때 참고하시면 될 것 같습니다.


3. heatmap_values.csv
   히트맵 각 셀의 값을 나타내는 파일입니다.
   각 셀이 포함한 입자의 개수를 값으로 가집니다.
   예를 들어, 제공드린 파일에서 0, 0 셀의 값은 1입니다.
   

4. SRU.csv
   수색 드론의 스팩을 나타내는 파일입니다.
   드론의 수색 경로 길이는 (수색 가능 시간 X 이동속도) 보다 같거나 작아야 합니다.
   
   저희 연구에서는수색 경로로부터 (탐지폭 / 2) 떨어진 곳에 히트맵 셀의 중심점이 위치한다면, 해당 셀을 포함한다고 정의했습니다.
   다만, 드론의 수색 범위가 작아서, 위와 같이 판단하기 어려울 수 있습니다. 해당 부분은 연구자분의 판단에 맡기겠습니다...

6. vertex.png
   경로의 이동을 나타내는 vertex에 대한 설명을 돕기 위한 이미지 파일입니다.
